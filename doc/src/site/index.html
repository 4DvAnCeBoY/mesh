Mesh is a graph powered reactive and clusterable CMS for the JVM.

{% icon "install" %} Download

Reactive

Mesh is based upon the non blocking Vert.x framework and developed using reactive patterns.

Mesh is fun

Enjoy being a developer again. Spare a thought for your less fortunate colleagues using legacy technologies.

  • Mesh is fast.

  • Mesh is an ideal choice for creating light-weight, high-performance, sites.

  • Mesh is not an application server. There's no monolithic Mesh instance into which you deploy applications. You just run your apps wherever you want to.

  • Mesh is modular - when you need more bits just add the bits you need and nothing more.

  • Mesh is simple but not simplistic. Mesh allows you to create powerful sites, simply.

Mesh is a CMS for creating compelling, 21st century sites.
